# PasteStore
LocalPaste is a lightweight web application that lets you save text and code snippets directly in your browser using local storage. No login, no server, and no internet required—your data stays private and accessible anytime on your device.

LocalPaste is a client-side paste management web application that stores pasted text locally in the browser using localStorage. It allows users to quickly add, view, and manage paste items without sending any data to a backend server.

The application works completely offline after loading and ensures full privacy, as all data remains on the user’s device. It is ideal for temporarily saving notes, code snippets, commands, or any text that needs quick access during development or daily work
